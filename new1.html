Edit the message sent from the treasure


Prepare the edit the code on the new micro:bit
  1. Unplug your original micro:bit from the computer.
  2. Unplug the battery back from the treasure micro:bit, and put the battery onto your original micro:bit.
  3. Plug the new, treasure, micro:bit into the computer.

Make your changes to the code
  1. Open the starter program and click edit.
  2. Edit the radio set group block to be your computer's number, to make sure that your two micro:bits can communicate with each other.
  3. Leave the power level at 1 - this tells the treasure to transmit measures using as low a power as possible, so that it isn't too easy to find the treasure.
  4. Edit the message the micro:bit is sending.
  5. Test your new code - can your original micro:bit still find the treasure?

Make more changes

You can make other changes to this code, for example editing the pictures - try creating an animation.

The treasure micro:bits do not have speakers on, so you cannot play music or sound effects on this micro:bit.
