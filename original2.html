Replace the bar graph with pictures


In the Logic menu, find the if true block.

Put it inside your on radio received event, at the end after all of the code you used before.



Also in the Logic menu, find the comparison block 0 = 0.

Put it on the if true block, so that it replaces true.



Edit this new block, so that your statement reads: if signal-strength = 0.


  • signal-strength is in the Variables menu
  • = means is equal to
  • 0 is the lowest possible signal strength

You can add something into the space inside the if block now. Whatever you add inside, will happen only if the signal strength is exactly 0.



To be able to test this code properly - delete the bar graph code you used earlier.

Right click on the purple block, and click 'delete blocks'.



Click the + button at the end of the if block twice, to expand the block.

Repeat the instructions above, to add a new set of instructions for if the signal strength is equal to 1.


You can now have three options:

  • if the signal strength is 0
  • if the signal strength is 1
  • if the signal strength is anything else

Once you have tested this code, you can keep going (keep pressing the + button!) to add instructions for each signal strength all the way up to 9.
